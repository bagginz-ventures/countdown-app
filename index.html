How to

Use this countdown app

Features

  • Shows a live countdown to your saved target date and time.
  • Saves one countdown on this device.
  • Optionally shows a notification when the countdown finishes.
  • Supports backup export and import.

Main buttons

  • Save countdown — saves the current label, date, time, notes, and notification setting.
  • Clear — removes the saved countdown from this device.
  • Compact view — opens the simplified widget-style view.
  • Export backup — downloads the current countdown as JSON.
  • Import backup — restores a previously exported countdown file.
  • Copy compact link — copies a direct link to the compact view.

Compact widget view

  • Shows the countdown in a cleaner, smaller layout.
  • Best for keeping visible in a narrow always-open window.
  • Hides the form and extra controls so the countdown stays front and center.

This web app is obsolete. It is retained only as a reference build and should not be used as the active product direction.

Use it like an app

Tip: the compact view is good for a pinned browser tab, a tiny app window, or an Android shortcut.

  • Each device stores its own countdown locally.
  • Data persists across refreshes, reboots, and offline use.

  • For multi-year safety, export and import backup support is included.
